1. Objetivo del proyecto

El objetivo de este proyecto es analizar las ventas de un negocio utilizando técnicas de minería de datos con apoyo de inteligencia artificial, con el fin de identificar los factores que influyen en el total de ventas y reconocer patrones de comportamiento entre los productos.

2. Pregunta de investigación

¿Qué factores influyen más en el total de ventas de un negocio y qué patrones de venta se pueden identificar según la categoría de los productos?

3. Datos utilizados

Se trabajó con un conjunto de datos de creación propia compuesto por 20 registros de ventas.
El dataset incluye información sobre fecha de venta, producto, categoría, precio unitario, cantidad vendida, canal de venta, costo unitario, venta total y margen.

Los datos fueron simulados con fines académicos y estructurados en Excel.

4. Preparación y limpieza de los datos

Con apoyo de herramientas de inteligencia artificial, los datos fueron revisados y preparados en Excel.
El proceso incluyó la verificación de valores faltantes, revisión de duplicados, corrección de formatos de fecha y validación de variables numéricas.
Posteriormente, se trabajó con una versión final limpia del dataset.

5. Técnicas de minería de datos aplicadas
Análisis descriptivo

Se utilizaron tablas dinámicas para analizar el comportamiento de las ventas por categoría y por canal de venta.

Regresión lineal

Se aplicó un modelo de regresión lineal utilizando el complemento Data Analysis ToolPak de Excel.
La variable dependiente fue el total de ventas, mientras que las variables independientes fueron el precio unitario, la cantidad vendida y el costo unitario.

Segmentación

Se realizó una segmentación simple de los productos con base en el total de ventas, identificando grupos de bajo, medio y alto rendimiento.

6. Resultados y visualización

Los resultados muestran que la cantidad vendida es el factor con mayor influencia en el total de ventas, seguido por el precio unitario.
La categoría Tecnología concentra el mayor volumen de ventas, mientras que las categorías Ropa y Alimentos presentan menores ingresos relativos.

Las visualizaciones (gráficos de barras, dispersión y líneas) facilitaron la interpretación de los datos y permitieron identificar patrones claros de comportamiento.

7. Conclusiones

El análisis evidencia que el volumen de ventas tiene un mayor impacto en los ingresos que el precio por sí solo.
Asimismo, el uso de visualizaciones y regresión permitió obtener insights claros que pueden apoyar la toma de decisiones comerciales.

8. Limitaciones

El análisis se basa en un conjunto reducido de datos simulados, por lo que los resultados no deben generalizarse sin un mayor volumen de información y datos reales.

9. Uso de inteligencia artificial
